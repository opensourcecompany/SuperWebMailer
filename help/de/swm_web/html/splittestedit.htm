A/B Split Test bearbeiten

 

Empfängerliste

 

Name des A/B Split Tests Vergeben Sie eine eindeutige Bezeichnung z.B. Vergleich Newsletter-Varianten.

 
Beschreibung Optional können Sie einen Beschreibung angeben.

 
Empfängerliste Wählen Sie die Empfängerliste aus, die für den A/B Split Test verwendet werden soll. In der weiteren Einrichtung des A/B Split Tests werden nur E-Mailings für den Test angezeigt, die diese Empfängerliste verwenden.

Sollten Sie die Empfängerliste eines bestehenden Split Tests ändern, dann werden alle Statistiken dieses Split Tests dauerhaft gelöscht.

 

 

E-Mailings

 

 

E-Mailings für den A/B Split Test Wählen Sie mindestens 2 E-Mailings für den A/B Split Test aus.

Hinweis: Es werden nur E-Mailings angezeigt, für die die gleichen Regeln für den E-Mail-Versand, der Status "Nur speichern" gewählt und das anonyme Tracking (Zählung der Öffnungen und Zählung Klicks auf Hyperlinks) aktiviert worden ist.

 

 

Einstellungen

"Gewinner" des A/B Split Test bestimmen Wählen Sie aus wie der Gewinner des Split Tests bestimmt werden soll. Bei der Variante "anhand der Klicks auf Hyperlinks in der E-Mail" werden alle Klicks auf getrackte Hyperlinks in der E-Mail summiert, hingegen bei der Variante "anhand der Öffnungen der E-Mail" werden nur die reinen E-Mail-Öffnungen summiert.

 
Versand an alle Empfänger der Empfängerliste Die Empfänger der Empfängerliste werden zufällig ausgewählt und in gleichen Gruppen mit den gewählten E-Mailings gesendet.

z.B. enthält die Empfängerliste 100 Empfänger und es wurden 2 E-Mailings gewählt, dann wird an 50 zufällig ausgewählte Empfänger das erste E-Mailing und an die übrigen 50 Empfänger das zweite E-Mailing versendet.

 
Versand an einen Teil der Empfänger der Empfängerliste, anschließend wird die "Gewinner"-E-Mail an die übrigen Empfänger gesendet Wählen Sie aus wie viele Prozent der Empfänger in der Empfängerliste die gewählten E-Mailings erhalten sollen. Nach dem Versand der E-Mailings wird die angegebene Zeit gewartet bis der Gewinner des Split Tests bestimmt und danach das Gewinner-E-Mailing an den Rest der Empfänger in der Empfängerliste gesendet wird.

 

 

Planung

 

A/B Split Test nur speichern Speichert den A/B Split Test, dieser wird nicht versendet.

 
A/B Split Test nach Speicherung der Einstellungen versenden (CronJob) Speichert den A/B Split Test und versendet diesen sofort.

 
A/B Split Test zu einem späteren Zeitpunkt versenden (CronJob) Speichert den A/B Split Test und versendet diesen zum angegebenen Zeitpunkt.

 

 

Zusammenfassung

Auf der Seite der Zusammenfassung werden alle Einstellungen nochmals dargestellt. Klicken Sie auf "Fertig stellen" um die Einstellungen zu speichern und den A/B Split Test, falls gewählt, versenden zu lassen.
