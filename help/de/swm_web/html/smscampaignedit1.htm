SMS-Kampagne ändern

Allgemein/Empfängerliste

 

Name der SMS-Kampagne Geben Sie Ihrer SMS-Kampagne einen neuen Namen z.B. Kunden-Einladung.

 
Beschreibung Sie können optional eine Beschreibung eingeben.

 
Empfängerliste für den Versand der SMS Wählen Sie die Empfängerliste welche für den E-Mail-Versand genutzt werden soll.

Hinweise:

  • Sollten Sie bei einer bestehenden SMS-Kampagne die Empfängerliste ändern, dann wird die Statistik der SMS-Kampagne komplett und unwiederbringlich gelöscht.
  • Es wird nur versucht SMS an die Empfänger zu senden, die im Feld mit der Mobilfunknummer einen Wert enthalten. Dieser Wert wird vor dem eigentlichen SMS-Versand geprüft, damit werden nur SMS an die Empfänger gesendet, die eine syntaktisch gültige Mobilfunknummer enthalten.
