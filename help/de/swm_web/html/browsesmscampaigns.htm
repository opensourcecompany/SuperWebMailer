SMS-Kampagnen

Verwenden Sie SMS-Kampagnen, um personalisierte Serien-SMS oder unpersonalisierte Massen-SMS an die Empfänger der gewählten Empfängerliste versenden zu lassen. Es werden alle SMS-Kampagnen dauerhaft gespeichert, so dass Sie diese mehrfach oder nach Modifikation des SMS-Inhalts versenden lassen können.

Hinweis: Für den Versand von Serien-SMS/Massen-SMS ist ein Konto beim SMS-Dienstleister SMSout.de notwendig, es kann kein anderer Anbieter verwendet werden.

 

 

Spalte "Versendet"

In der Spalte wird angezeigt, wann die SMS-Kampagne das letzte Mal versendet worden ist. Eine nicht fertig gestellte oder niemals versendete SMS-Kampagne besitzt immer als Versanddatum "niemals".

 

Aktionen mit SMS-Kampagnen ausführen

Filter anzeigen / nicht anzeigen Mit Hilfe des Filters können Sie nur bestimmte SMS-Kampagnen suchen und anzeigen lassen. Ebenfalls beeinflussen Sie mit Hilfe des Filters die Reihenfolge der Anzeige der SMS-Kampagnen und die Anzahl SMS-Kampagnen, die auf einer Seite angezeigt werden sollen.

 
Neue SMS-Kampagne Ermöglicht es eine neue SMS-Kampagne zu erstellen.

 
Symbol Ermöglicht das Ändern einer SMS-Kampagne.

 
Symbol Löscht dauerhaft die SMS-Kampagne und alle gespeicherten Statistiken.

 
Symbol Ermöglicht es eine Test-SMS zu versenden.

Hinweis
Nur wenn im Assistenten zur Bearbeitung der SMS-Kampagne auf "Fertig stellen" geklickt wurde, ist diese Symbol vorhanden.

 
Symbol Zeigt das Versand-Protokoll der SMS-Kampagne.

Hinweis
Nur wenn im Assistenten zur Bearbeitung der SMS-Kampagne auf "Fertig stellen" geklickt wurde und die SMS-Kampagne versendet worden ist, wird dieses Symbol dargestellt.

 
Auswahlboxen Sie können mehrere SMS-Kampagnen mit Hilfe der Auswahlboxen selektieren und auf die selektierten SMS-Kampagnen eine gemeinsame Aktion anwenden.

Die möglichen Aktionen sind

SMS-Kampagne(n) löschen Löscht dauerhaft alle SMS-Kampagne(n) und alle gespeicherten Statistiken.

 
SMS-Kampagnen-Einstellungen duplizieren Ermöglicht es ein oder mehrere SMS-Kampagne(n) zu duplizieren.
Es werden dabei alle Einstellungen dupliziert, jedoch nicht die Versand-Einträge der SMS-Kampagne. Die SMS-Kampagne selbst erhält nach dem Duplizieren den Status "nicht versendet" und beim Bearbeiten der SMS-Kampagnen-Einstellungen muss nochmals der Assistent durchlaufen werden.
