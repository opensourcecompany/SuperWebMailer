Importieren von Empfängern

Es können Empfänger aus einer CSV-Datei (Text mit Trennzeichen) oder aus einer anderen Datenbank importiert werden.

 

Auswahl der Quelle

Wählen Sie "Aus CSV-Datei (Text mit Trennzeichen) importieren" um die Empfänger aus einer Datei importieren zu lassen. Die Datei kann vorher per FTP oder Upload übertragen werden.
Die Variante "Aus externer Datenbank importieren" ermöglicht es Empfänger aus einer anderen Datenbank direkt zu importieren. Für den Import sind die Zugangsdaten für diese Datenbank notwendig.
