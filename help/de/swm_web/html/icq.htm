So bekommen Sie das Programm ICQ

Sie erhalten das Programm ICQ auf der Web-Site http://www.icq.com. Das Programm ist in englischer Sprache und kann nur verwendet werden, wenn Sie Ihre Internet - Verbindung über das DFÜ - Netzwerk aufbauen bzw. bei einer Standleitung zum Internet.

Mit Hilfe dieses Programms können Sie mit uns kommunizieren. Störungen durch andere Leute sind ausgeschlossen, da jeder Nutzer eine eigene (einmalige) ID erhält, nur wer selbst ein ID besitzt und meine ID kennt, kann sich mit mir in Verbindung setzen.
