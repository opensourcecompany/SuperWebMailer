SMS-Kampagne fertiggestellt

Haben Sie die Variante "SMS-Kampagne sofort live versenden" gewählt, dann können Sie auf "SMS-Versand jetzt beginnen" klicken, um sofort die SMS im geöffneten Browserfenster versenden zu lassen.

Wurde der SMS-Versand per CronJob sofort oder zu einem späteren Zeitpunkt definiert, dann wird das CronJob-Script die SMS versenden.

Haben Sie die Variante "SMS-Kampagne nur speichern" gewählt, dann können Sie weitere Änderungen vornehmen oder zur Übersicht der SMS-Kampagnen zurückkehren.
