SMS-Inhalt

Optional, Name der SMSout SMS-Kampagne Es kann der Name einer SMS-Kampagne eingegeben werden. Im SMSout-Kundenbereich in der Statistik können Sie eine Aufstellung der Anzahl SMS und Kosten pro SMS-Kampagne abrufen.

 
Text der SMS Geben Sie den Text der SMS ein, beachten Sie dabei, dass die SMS mit einem Buchstaben beginnen sollte, um Probleme bei der Anzeige der SMS beim Empfänger zu vermeiden.

 

Platzhalter verwenden, um den SMS-Empfänger persönlich anzusprechen

  • In der Auswahlliste "Serien-SMS-Feld" können Sie Platzhalter wählen, die beim Versand der SMS durch die Daten des Empfängers ersetzt werden sollen.
  • Klicken Sie auf die Schaltfläche um eine Funktion z.B. für die persönliche Anrede mit einem Klick auf die Schaltfläche einfügen zu lassen.
  • Um eine definierte E-Mail-Vorlage (nur reiner Text) zu laden klicken Sie auf die Schaltfläche und wählen Sie die Vorlage aus. Angezeigt werden im Dialog mit den Vorlagen nur Vorlagen des Typs Text.

 

 
Eingegebene Zeichen Anhand der zuvor gewählten Versandvariante werden die Anzahl Zeichen pro SMS angezeigt. Sollte die Zeichenanzahl der zu versendenden SMS eine SMS überschreiten, dann wird automatisch der SMS-Text in mehrere SMS am Zeichen 159 aufgeteilt.
