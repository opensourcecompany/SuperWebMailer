E-Mailing fertiggestellt

Haben Sie die Variante "E-Mailing sofort live versenden" gewählt, dann können Sie auf "E-Mail-Versand jetzt beginnen" klicken, um sofort die E-Mails im geöffneten Browserfenster versenden zu lassen.

Wurde der E-Mail-Versand per CronJob sofort oder zu einem späteren Zeitpunkt definiert, dann wird das CronJob-Script die E-Mails versenden.

Haben Sie die Variante "E-Mailing nur speichern" gewählt, dann können Sie weitere Änderungen vornehmen oder zur Übersicht der E-Mailings zurückkehren.
