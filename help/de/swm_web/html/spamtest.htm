Spam-Test

Führt mit Hilfe des auf dem Server oder externen Server installierten SpamAssassin einen Spam-Test durch. Getestet wird dabei der Inhalt der E-Mail auf spamverdächtige Merkmale z.B. bestimmte Farben, Schriftformatierungen und Worte. Ziel des Spam-Tests sollte es sein, so wenig wie nur möglich Spam-Punkte zu erreichen.

Eine weitere Beschreibung der Abkürzungen finden Sie unter http://www.supermailer.de/spamassassin.htm.
