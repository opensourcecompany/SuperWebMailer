Meldungstexte

Die Meldungstexte werden den An-/Abmeldeformularen zugewiesen. Es können verschiedene Meldungstexte definiert werden z.B. um mehrsprachige An-/Abmeldeformulare zu erstellen.

 

Aktionen ausführen

Neue Meldungstexte anlegen Ermöglicht es ein neues Meldungstexte-Set zu erstellen.

 
Symbol Ermöglicht das Ändern eines bestehenden Meldungstexte-Sets

 
Symbol Löscht dauerhaft das Meldungstexte-Set.

Hinweis:
Das Löschen ist nur möglich, wenn es kein Standard Meldungstexte-Set und nicht einem An-/Abmeldeformular zugewiesen ist.
