E-Mailing/Newsletter ändern

Allgemein/Empfängerliste

 

Name des E-Mailings Geben Sie Ihrem E-Mailing einen neuen Namen z.B. Newsletter Januar 2009.

 
Beschreibung Sie können optional eine Beschreibung eingeben.

 
Empfängerliste für den Versand der E-Mails/Newsletter Wählen Sie die Empfängerliste welche für den E-Mail-Versand genutzt werden soll.

Hinweis: Sollten Sie bei einem bestehenden E-Mailing die Empfängerliste ändern, dann wird die Statistik des E-Mailings komplett und unwiederbringlich gelöscht.
