Meldungstexte bearbeiten

Es können für die An-/Abmeldeformulare einer Empfängerliste verschiedene Formulare mit verschiedenen Meldungstexten definiert werden, die im Erfolgsfall oder im Fehlerfall ausgegeben werden. Die Meldungstexte können z.B. in verschiedenen Sprachen definiert werden, so dass An-/Abmeldeformulare in verschiedenen Sprachen realisiert werden können. Alle Meldungstexte können als reiner Text oder als HTML definiert werden, etwaige Platzhalter sind in diesen Texten nicht verwendbar.

Hinweis
Die Meldungstexte gelten global für alle Empfängerlisten, die ein Administrator und etwaige definierte Nutzer erstellen kann.
