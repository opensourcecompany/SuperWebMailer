PRODUCTAPPNAME - Verzeichnisse festlegen
Das Installationsscript hat versucht die Verzeichnisangaben selbständig zu bestimmen. Bitte prüfen Sie genau diese Angaben auf Richtigkeit, ansonsten wird die Ausführung dieses Scriptpakets fehlschlagen.


Prüfen Sie die Rechte auf die Dateien config_db.inc.php und config_paths.inc.php im Verzeichnis in dem sich ebenfalls dieses Script befindet. Unter Linux/Unix-Systemen müssen die Rechte der beiden Dateien auf 777 (ALLE LESEN, ALLE SCHREIBEN, ALLE AUSFÜHREN, chmod 777) gesetzt werden.
Beispiel für das Festlegen der Rechte einer Datei mit dem FTP-Programm auf Linux-Systemen
Auf Windows-Servern müssen die Rechte so gesetzt werden, damit die Datei geändert werden kann (Vollzugriff empfehlenswert).
